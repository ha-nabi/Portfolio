## 🤔 Experiences
- Swift로 웹페이지 구축
  - SwifWeb을 사용하여 오직 Swift로만 코드를 작성하여 웹페이지 구축
  - WASM에 대한 미약한 공부
  - 간접적인 css 사용법

<br>

## 🏃 Run Project (Local)
```sh
$ brew install swifweb/tap/webber
$ webber serve
```
